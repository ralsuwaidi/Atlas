
# Atlas

## Usage 

```
pip install -r requirememts
docker compose build
docker compose up
python app.py
```

## Connect to db

1. log into `http://localhost:3300`
2. click create new
3. type `mongodb` in host then click save
4. click on new rown then click connect

## Connect to flask

2. click games
